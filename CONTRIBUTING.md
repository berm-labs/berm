# Contributing to BERM

Thanks for your interest in BERM, the parametric cover protocol on Solana.

## Getting started

Clone the repository and build the workspace:

```bash
git clone https://github.com/berm-labs/berm.git
cd berm
cargo fmt --all
```

## Guidelines

- Keep changes focused and well described.
- Match the existing formatting (`cargo fmt`, `prettier`).
- Open an issue before large changes so we can discuss the approach.

## Reporting issues

Use GitHub issues for bugs and feature requests. Include reproduction
steps and the cluster you observed the behaviour on.
